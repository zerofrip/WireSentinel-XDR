# WireSentinel-XDR

Extended Detection and Response (XDR) platform for the WireSentinel ecosystem.

## Components

- **EDR** — Endpoint Detection and Response
- **NDR** — Network Detection and Response
- **ITDR** — Identity Threat Detection
- **Hunting** — Threat hunting platform
- **Detections** — Detection engineering
- **Incidents** — Incident management
- **Cases** — Case management
- **SOAR** — Security orchestration automation
- **Attack Graph** — Attack path analysis
- **MITRE** — ATT&CK mapping
- **Response** — Automated response actions

## Build

```bash
cargo test --workspace
```
